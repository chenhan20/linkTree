Strava API 接入方向

① 取得 Access Token（OAuth 2.0）
到 strava.com/settings/api 建立 App，拿到 client_id 和 client_secret。
用瀏覽器跑一次 OAuth flow → 換到 refresh_token（只需做一次）。

② 定期刷新 token + 抓資料
POST https://www.strava.com/oauth/token 換新的 access_token（6小時有效）。
GET https://www.strava.com/api/v3/athlete/activities 拿最近活動。
GET https://www.strava.com/api/v3/athletes/{id}/stats 拿總統計。

③ 你可以這樣串
用你熟悉的 Node.js + Express 寫一支 API server，定時刷 token 存到 DB 或 JSON 檔，前端 Vue fetch 你自己的 endpoint，這樣不用把 secret 暴露在前端。

④ 更懶的方式
用 GitHub Actions 排程每天跑 script 更新 strava.json 推到 repo，GitHub Pages 前端直接 fetch raw JSON。零 server 成本。

ITT Segment Card · Redesign Exploration

5 個方向，5 種炫炮

不管效能、只求視覺衝擊。每個版本對應一個 2026 現正流行、評價很好的實驗性互動手法，套用同一組真實 ITT 資料（風櫃嘴 / 中社路 / 社子島砍鴨頭 / 劍南路回程 / 碧山露營場）。滑動滑鼠 hover 每張卡片感受差異，選一個方向我再收斂做成正式版。

01 / 05
全息卡牌 HOLO FOIL
靈感來自 Pokémon 卡牌全息效果 的 2026 復刻版：JS 追蹤滑鼠算出 --mx/--my，驅動 3D 傾斜(rotateX/Y) + 對角彩虹全息貼膜(mix-blend-mode:color-dodge) + 邊框流光。滑鼠移到哪，卡片就往哪傾斜、彩虹光斑就跟到哪，像真的在轉動一張限定卡。
02 / 05
液態玻璃 LIQUID GLASS
2026 年最被討論的介面語言：Apple Liquid Glass 的網頁版詮釋。多層 backdrop-filter 折射出背後漂浮的色塊，卡片邊緣有一條會呼吸的高光線，hover 時整張卡浮起、滑鼠位置變成一道會移動的鏡面反光，質感比舊版玻璃擬態更真實、更有深度層次。
03 / 05
賽博 HUD CYBERPUNK HUD
切角外框 + 掃描線 + 系統終端機字體，hover 觸發故障閃爍(RGB 色差)，滑鼠在卡片內留下瞄準十字準星，像 FPS 遊戲 HUD 鎖定目標。呼應賽博朋克儀表板一直是資料視覺化很受歡迎的風格方向。
04 / 05
動態字體 Bento KINETIC BENTO
Bento Grid 2.0：不等大小的模組化排版，一眼掃過 5 段成績；數字用超大字重當主視覺，卡片有彈簧回彈的「squish」手感，按鈕會被滑鼠磁吸靠近。2026 年 Q3 檢視後仍然是留存率最高的兩個趨勢之一。
05 / 05
流體漸層 FLUID SHADER
每張卡背景是一顆用 Canvas 即時運算的流體色塊（模擬 WebGL shader 質感，純 2D API 不吃額外套件），色彩取自路段 accent；hover 會炸開一小波粒子慶祝，呼應近年 Awwwards 常見的「WebGL 流動背景＋cinematic reveal」手法。

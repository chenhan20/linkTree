## SteveChuang Link Tree

使用 GitHub Pages 靜態部署的個人入口網站，Vue 3 SPA 三頁架構。

### 功能
- 首頁：太空主題動畫（Canvas 星空、火箭、流星、粒子）
- Resume 頁：互動式線上履歷（Experience、Projects、Skills）
- Strava 頁：自行車數據儀表板（里程、月份趨勢、最近活動）

### 技術
- Vue 3 (CDN, Composition API)
- Canvas API（星空背景）
- CSS 動畫
- Strava API（透過 GitHub Actions 定期抓取寫入 strava.json）
